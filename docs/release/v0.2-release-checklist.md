The integrated release branch stages 67 production runtime files. Production
staging and release-safety validation pass. Source and staged XML parse with
`xmllint`. The staged production package contains 19 unique `defName` values with
no duplicates; the staged developer package contains 7 unique `defName` values
with no duplicates and automatically discovers 18 development resource entries.
All staged `texPath`, `immatureGraphicPath`, and `uiIconPath` references resolve.
The staged content-manifest SHA-256 is
`9dcbcebeeb3002e6086d1cef9439515688b6daf102c46db01edcbf92015c379d`.

## Candidate Manual Evidence

Patrick Mee completed a playthrough using the exact candidate staged into
RimWorld's local `Mods/EmeraldIsle` directory and reported that it looked good on
2026-07-13. This closes the clean-load and representative FS-006/FS-007 gameplay
check. Patrick Mee also confirmed that the playthrough covered save/reload and the
