## Summary

The Hearth and Household expands the first holding with two small domestic
systems: a flax-to-linen textile path and a central hearth. It keeps the mod
XML-only and vanilla-first while adding visible household labor, clothing, heat,
light, and cooking identity.

## Included Content

- All Version 0.1 content: oats, dry-stone wall, hand quern, milled oats, oat
  porridge, and oat flatbread.
- Flax: a ground-grown fiber crop that matures faster than cotton but requires
  processing before use.
- Raw flax and linen: harvested flax is processed into linen at vanilla work
  locations; linen remains a general Fabric textile with a warm-weather niche and
  lower durability than cloth.
- Linen tunic: a linen-only everyday garment inspired by the `leine` category,
  using an English player-facing name and original runtime art.
- Central hearth: a stone-stuffable, continuously fueled household hearth that
  provides heat, light, gathering-spot behavior, campfire-grade cooking bills, and
  Emerald Isle oat-food bills through vanilla XML behavior.

The release contains no C#, Harmony patches, required DLC, custom social system,
Irish-language player text, new fuel resource, flax oil, brat cloak, loom,
spindle, dairy chain, feast system, or broader furniture set.

## Installation

Subscribe on the Steam Workshop for automatic installation after publication, or
install manually:

1. Copy the staged `EmeraldIsle` folder into RimWorld's local `Mods` directory.
2. Confirm the resulting path contains `EmeraldIsle/About/About.xml` rather than
   an additional nested `EmeraldIsle` folder.
3. Enable **Emerald Isle** after **Core** in the RimWorld Mods list and restart
   when prompted.

The developer-testing package is separate and must not be installed for normal
play or included in a public release archive.

## Upgrade and Existing Saves

Adding Version 0.2.0 to a Version 0.1 save is expected to make flax, linen, the
linen tunic, and the central hearth available without migration or custom
serialized state.

Back up important saves before changing any mod list. Removing Emerald Isle from
a save that contains its plants, items, buildings, blueprints, bills, food-policy
references, stockpile filters, outfits, inventories, or caravan contents can
produce ordinary missing-definition warnings and remove affected content. Consume
or remove mod content first where practical, save a backup, and expect content
loss.

## Compatibility and Intended Behavior

- Core-only operation is required; supported DLC add no Version 0.2-specific
  behavior.
- Linen participates in vanilla Fabric behavior. The linen tunic is intentionally
  linen-only.
- Flax has a shorter field commitment than cotton but adds processing labor and
  does not support hydroponics.
- The central hearth is a larger, slower, continuously fueled household fixture.
  It should not replace the fueled stove as the best compact kitchen tool.
- The hearth includes Emerald Isle oat porridge and oat flatbread bills so it can
  support the existing oat-food loop.
