# v0.0.0: The Studio Exists

**Release date:** 2026-07-04  
**Scope:** Milestone 0 and Milestone 0.5  
**Gameplay content:** None

## Summary

Emerald Isle now has the studio foundation required for long-term development by
human contributors and AI coding agents. This release contains governance,
research, design, engineering, art, quality, compatibility, and release practices.
It intentionally contains no RimWorld package, XML, C#, textures, audio, or
gameplay systems.

## Milestone 0: Build the Studio

- Ratified the Constitution and authority hierarchy.
- Defined vision, architecture, documentation strategy, balance, art, coding, XML,
  C#, asset, localization, research, QA, release, Git, versioning, and AI practices.
- Adapted Spec Kit templates and gates for RimWorld feature development.
- Added contribution, governance, conduct, security, support, onboarding, issue
  triage, Definition of Ready, Definition of Done, and feature lifecycle guidance.
- Added ADR, feature, research, playtest, release, and postmortem templates.
- Added build, performance, RimWorld modding, DLC/mod, and save-compatibility policies.
- Added project risk, technical-debt, research, and glossary memory.

## Milestone 0.5: Create the Design Bible

- Established the canonical Design Bible and Feature Acceptance Checklist.
- Defined project identity, exclusions, canon, cultural transformation, language,
  mythology, progression, architecture, performance, compatibility, and expansion rules.
- Expanded the product roadmap through Version 3.0 with milestone contracts.
- Created the official candidate Feature Catalog with 43 discovery-stage candidates.

## Final Project Decisions

- Source code and source-code tooling: MIT License.
- Creative assets and writing: separately licensed; see `CREATIVE_ASSETS_LICENSE.md`.
- Required platform: RimWorld 1.6 Core.
- Officially supported DLC: Royalty, Ideology, Biotech, Anomaly, and Odyssey.
- Nine foundation authorities are stable and normally require an ADR for material changes.

## Validation

- Internal Markdown links resolved.
- All required backlog categories were present.
- Repository scan found no gameplay XML, C#, DLLs, art, textures, icons, or audio.
